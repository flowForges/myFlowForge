# 粉色猫娘

明确成年、粉白运动风的动态猫娘桌面宠物。角色为约 6.5 头身，长粉发、猫耳与长尾；五种状态保持同一角色、服装和配色。

- `idle`：面向屏幕交叉猫步，手臂、腰胯和尾巴随步伐换重心。
- `working`：左右踏跳舞蹈，包含腾空、落地和头发衣摆的跟随动作。
- `confirm`：向屏幕前倾招手，随后指向确认区域。
- `input`：向屏幕招手，随后用左手邀请填写并轻点输入区域。
- `done`：蓄力、跳起、双手庆祝、落地回弹。

运行时优先使用 `webp/` 中的透明动画；根目录 GIF 用作兼容输出，`apng/` 保存透明 APNG，`png/` 保存设置页或降级场景使用的静态帧。`frames/` 和 `source/` 保留可复现的关键帧及精灵表。
